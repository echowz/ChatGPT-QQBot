# ChatGPT-QQBot
运行在qq即时聊天通讯应用程序上的chatgpt对话式机器人

需要python3.7或以上版本环境，cqhttp框架、NoneBot框架支持。\
可参考NoneBot快速入门指南：https://v1.nonebot.dev/guide/installation.html
